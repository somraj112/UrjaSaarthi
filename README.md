# UrjaSaarthi

> AI-powered platform to simplify solar adoption for households.

---

## What is UrjaSaarthi?

UrjaSaarthi helps users make **smart, cost-effective solar decisions** by analyzing their electricity usage and providing clear, personalized recommendations.

It turns complex energy calculations into **simple, actionable insights** — so users can confidently decide whether going solar makes sense for them.

---

## What it does

*  Analyzes electricity bill & household usage
*  Recommends optimal solar system size
*  Estimates savings, ROI, and investment
*  Visualizes long-term benefits
*  Suggests trusted solar providers

---

## Tech Stack

* **Frontend:** React + TypeScript
* **AI:** Groq-SDK
* **Backend APIs:** Groq API
* **Libraries:**

  * recharts (graphs & visualization)
  * jsPDF (report generation)
* **Others:** REST APIs

## Setup & Run Locally

### 1. Clone the repo

```bash
git clone https://github.com/your-username/urjasaarthi.git
cd urjasaarthi
```

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
```

### 4. Run the project

```bash
npm run dev
```

---

## Notes

* Avoid exposing API keys in frontend
* Free API tiers have rate limits
* Use backend proxy for production
